# Kurisu-bot
An example discord bot made with Skript and Vixio.

## Dependencies

 * Vixio 1.1.5
 * skript-mirror 0.7.1
 * Skript

## Commands

Command | Aliases | Description 
------------ | ------------- | -------------
say | reply | Makes Kurisu reply with the specified message
